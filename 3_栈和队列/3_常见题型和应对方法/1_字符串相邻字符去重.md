# 字符串相邻字符去重（Leetcode - 1047）

## 问题要点

给出由小写字母组成的字符串 `s`，**重复项删除操作**会选择两个相邻且相同的字母，并删除它们。

在 `s` 上反复执行重复项删除操作，直到无法继续删除。

在完成所有重复项删除操作后返回最终的字符串。答案保证唯一。

## 核心思想和套路 - 栈

### 如果当前元素和栈顶元素相同

栈顶元素出栈，当前元素不入栈

### 如果当前元素和栈顶元素不同

栈顶元素不出栈，当前元素入栈

## 实现代码

```java
class Solution {
    //类似于消消乐
    //栈 + 贪心算法？
    public String removeDuplicates(String s) {
        Stack<Character> stack = new Stack<>();
        for(char curr : s.toCharArray()){
            if(!stack.isEmpty()){
                char top = stack.peek();
                    if(curr == top){
                    stack.pop();
                }else{
                    stack.push(curr);
                }
            }else{
                stack.push(curr);
            }
        }

        StringBuilder sb = new StringBuilder();
        while (!stack.isEmpty()) {
            char top = stack.pop();
            sb.insert(0, top);
        }
        return sb.toString();
    }
}
```
