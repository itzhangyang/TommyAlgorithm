# 3. **栈与队列**

## 常见题型：

- 括号匹配
- 单调栈问题（如下一个更大元素）
- 用栈实现队列或用队列实现栈

## 解题策略：

- **栈**：擅长处理具有“最近相关性”的问题。
- **单调栈**：适用于处理区间范围最大/最小值问题。
- **双端队列**：用于滑动窗口最大值问题。

## 基础概念对比

| 特性     | 栈（Stack）          | 队列（Queue）            |
| -------- | -------------------- | ------------------------ |
| 存取顺序 | 先进后出（LIFO）     | 先进先出（FIFO）         |
| 常用操作 | `push`, `pop`, `top` | `enqueue`, `dequeue`     |
| 应用场景 | 括号匹配、逆序处理等 | 广度优先搜索、任务调度等 |
