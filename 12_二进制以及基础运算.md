# 常见运算一览

| 运算符 | 名称         | 含义                         | 常见用途示例                                         |
| ------ | ------------ | ---------------------------- | ---------------------------------------------------- |
| `&`    | 按位与 AND   | 两位都为1时结果才为1         | 掩码提取某一位状态，如 `n & 1` 判断奇偶              |
| ` | `            | 按位或 OR                    | 只要有一个1结果就是1                                 ||
| `^`    | 按位异或 XOR | 不同为1，相同为0             | 消除重复值、求唯一元素，如 Leetcode 只出现一次的数字 |
| `~`    | 按位取反 NOT | 将0变1，将1变0               | 补码表示中用于取反（需要结合加1来得出负数）          |
| `<<`   | 左移         | 各位左移若干位，低位补0      | 相当于乘以 2 的若干次方，例如 `x << 3` = x × 8       |
| `>>`   | 有符号右移   | 各位右移若干位，高位补符号位 | 相当于除以 2 的若干次方（保留符号）                  |
| `>>>`  | 无符号右移   | 各位右移，高位补0            | 与 `>>` 类似，但忽略符号位                           |

## 怎样将十进制转化为二进制？

将十进制转化为二进制的基本算法是通过**不断除以2并记录余数**

```java
public static String decimalToBinary(int decimal) {
        StringBuilder binary = new StringBuilder();
        
        // 除2直到商为0
        while (decimal > 0) {
            binary.append(decimal % 2); // 记录余数
            decimal /= 2;  // 更新商
        }
        
        // 逆序输出
        return binary.reverse().toString();
    }

```

## 通过异或操作计算只出现一次的元素

```java
public int singleNumber(int[] nums) {
    int res = 0;
    // a ^ a = 0
    // a ^ 0 = a
    for (int num : nums){
        res ^= num;
    }
    return res;
}
```

## 按位异或: ^

### 异或的运算规则 - 相同为0，不同为1

- 两个位对应的位相同，结果是0；
- 两个位对应的位不同，结果是1；

### 异或运算的关键特性

- 相同的两个位会相互抵消 `1 ^ 1 = 0`, `0 ^ 0 = 0`
- 不同的两个位会变成 1:`1 ^ 0 =1`,`0 ^ 1 = 1`
- **对于任何数字 x, `x^x=0`**,`x ^ 0 =0`

## 按位与：&

### 按位与的规则 -  只有两个都为1时结果才为1

- 只有两个位都是1时，结果才位1
- 其他情况下结果都是0

# 什么是无符号数？

无符号整数是一种只能表示**非负整数**（即零和正整数）的数据类型。与有符号整数不同，无符号整数没有用于表示正负的符号位，因此它们能够表示的**最大值是相同位数有符号整数的两倍**。

### 无符号整数的工作原理

在计算机系统中，数字通常以二进制形式存储。对于有符号整数，最高位的二进制位通常用作符号位（0表示正数，1表示负数）。

然而，**无符号整数将所有位都用于表示数值的大小**。

**举个例子：**

- **8位有符号整数**：最高位用于表示符号，剩下的7位用于表示数值。这意味着它可以表示从 -128 到 127 的数字。
- **8位无符号整数**：所有8位都用于表示数值，因此它可以表示从 0 到 255 的数字。

### 无符号整数的常见用途

- **计数和索引**：当你知道某个值永远不会是负数时，比如数组的长度、循环的次数或者文件的大小。
- **内存地址**：内存地址通常是无符号的，因为它们指向内存中的位置，没有负的地址。
- **位操作**：在进行位操作时，如设置或清除特定位，无符号整数可以避免符号位带来的复杂性。
- **哈希函数和校验和**：这些算法通常涉及对数据的数学操作，结果是无符号的。
